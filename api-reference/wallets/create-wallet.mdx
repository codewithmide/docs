---
title: 'Create Wallet'
api: 'POST https://api.fossapay.com/v1/wallets'
description: 'Create a new wallet for a customer'
---

## Request

<ParamField body="type" type="string" required>
  Wallet type: `customer`, `merchant`, or `system`
</ParamField>

<ParamField body="customer_id" type="string">
  Your internal customer ID
</ParamField>

<ParamField body="customer_name" type="string" required>
  Customer or wallet name
</ParamField>

<ParamField body="customer_email" type="string" required>
  Customer email address
</ParamField>

<ParamField body="currencies" type="array" required>
  Array of currency codes to support (e.g., ["NGN", "USDT"])
</ParamField>

<ParamField body="metadata" type="object">
  Custom metadata
</ParamField>

### Request Example

```bash
curl -X POST https://api.fossapay.com/v1/wallets \
  -H "Authorization: Bearer fp_live_sk_xxxxxxxx" \
  -H "Content-Type: application/json" \
  -d '{
    "type": "customer",
    "customer_id": "cus_123",
    "customer_name": "John Doe",
    "customer_email": "john@example.com",
    "currencies": ["NGN", "USDT"],
    "metadata": {
      "user_id": "12345"
    }
  }'
```

```javascript
const wallet = await client.wallets.create({
  type: 'customer',
  customer_name: 'John Doe',
  customer_email: 'john@example.com',
  currencies: ['NGN', 'USDT']
});
```
