---
title: 'Get Payout Status'
api: 'GET https://api.fossapay.com/v1/payouts/{payout_id}'
description: 'Check the status of a payout'
---

## Path Parameters

<ParamField path="payout_id" type="string" required>
  The unique identifier of the payout
</ParamField>

## Request Example

```bash
curl https://api.fossapay.com/v1/payouts/pay_abc123 \
  -H "Authorization: Bearer fp_live_sk_xxxxxxxx"
```

```javascript
const payout = await client.payouts.getStatus('pay_abc123');
```

## Status Values

| Status | Description |
|--------|-------------|
| `pending` | Payout initiated, not yet processed |
| `processing` | Being processed by recipient bank |
| `completed` | Successfully completed |
| `failed` | Failed (funds refunded) |
