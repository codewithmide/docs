---
title: 'Bank Name Enquiry'
api: 'POST https://api-staging.fossapay.com/v1/transfers/fiat/bank-name-enquiry'
description: 'Perform bank account name enquiry'
---

## Request

<ParamField header="x-api-key" type="string" required>
  API Key for authentication
</ParamField>

<ParamField body="bankCode" type="string" required>
  The bank code for the account enquiry
</ParamField>

<ParamField body="accountNumber" type="string" required>
  The account number to enquire about
</ParamField>

### Request Example

```bash
curl -X POST https://api-staging.fossapay.com/v1/transfers/fiat/bank-name-enquiry \
  -H "x-api-key: your_api_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "bankCode": "00014",
    "accountNumber": "1232129310"
  }'
```

```javascript
const enquiry = await client.fiatTransfers.bankNameEnquiry({
  bankCode: '00014',
  accountNumber: '1232129310'
});
```
