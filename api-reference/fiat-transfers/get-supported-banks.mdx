---
title: 'Get Supported Banks'
api: 'GET https://api-staging.fossapay.com/api/v1/transfers/fiat/banks'
description: 'Get supported banks for transfers'
---

## Request

<ParamField header="x-api-key" type="string" required>
  API Key for authentication
</ParamField>

### Request Example

```bash
curl -X GET https://api-staging.fossapay.com/api/v1/transfers/fiat/banks \
  -H "x-api-key: your_api_key_here"
```

```javascript
const banks = await client.fiatTransfers.getSupportedBanks();
```
