## Overview

Retrieve detailed information about a specific customer using their unique customer ID.

## Request

<ParamField header="x-api-key" type="string" required>
  Your API key for authentication
</ParamField>

<ParamField path="customerId" type="string" required>
  The unique identifier of the customer
</ParamField>

## Response

<ResponseField name="status" type="string">
  Response status: `success` or `error`
</ResponseField>

<ResponseField name="statusCode" type="number">
  HTTP status code (200 for successful retrieval)
</ResponseField>

<ResponseField name="message" type="string">
  Human-readable response message
</ResponseField>

<ResponseField name="data" type="object">
  Customer object
</ResponseField>

## Example

<Tabs>
<TabItem value="curl" label="cURL">

```bash
curl -X GET https://api-staging.fossapay.com/api/v1/customers/fh8e2e1f-e253-23ac-b8ft-31ed2t7fb0df \
  -H "x-api-key: YOUR_API_KEY"
```

</TabItem>
<TabItem value="nodejs" label="Node.js">

```javascript
const customerId = 'fh8e2e1f-e253-23ac-b8ft-31ed2t7fb0df';
const response = await fetch(`https://api-staging.fossapay.com/api/v1/customers/${customerId}`, {
  method: 'GET',
  headers: {
    'x-api-key': 'YOUR_API_KEY'
  }
});

const customer = await response.json();
console.log(customer);
```

</TabItem>
</Tabs>

## Rate Limits

- Maximum 100 requests per minute per API key
- Burst limit of 20 requests per second
