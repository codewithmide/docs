---
title: 'Get Virtual Account'
api: 'GET https://api.fossapay.com/v1/virtual-accounts/{virtual_account_id}'
description: 'Retrieve details of a specific virtual account'
---

## Path Parameters

<ParamField path="virtual_account_id" type="string" required>
  The unique identifier of the virtual account
</ParamField>

## Request Example

```bash
curl https://api.fossapay.com/v1/virtual-accounts/va_abc123xyz \
  -H "Authorization: Bearer fp_live_sk_xxxxxxxx"
```

```javascript
const account = await client.virtualAccounts.get('va_abc123xyz');
```

```python
account = client.virtual_accounts.get('va_abc123xyz')
```

## Response

<ResponseField name="status" type="string">
  Response status: `success` or `error`
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="properties">
    <ResponseField name="virtual_account_id" type="string">
      Unique identifier for the virtual account
    </ResponseField>

    <ResponseField name="account_number" type="string">
      The virtual account number
    </ResponseField>

    <ResponseField name="account_name" type="string">
      Account name
    </ResponseField>

    <ResponseField name="bank_name" type="string">
      Providing bank name
    </ResponseField>

    <ResponseField name="bank_code" type="string">
      Bank code
    </ResponseField>

    <ResponseField name="type" type="string">
      Account type: `dedicated` or `temporary`
    </ResponseField>

    <ResponseField name="status" type="string">
      Account status: `active`, `inactive`, `expired`
    </ResponseField>

    <ResponseField name="total_received" type="number">
      Total amount received through this account
    </ResponseField>

    <ResponseField name="transaction_count" type="number">
      Number of transactions received
    </ResponseField>

    <ResponseField name="last_payment_at" type="string">
      Timestamp of last payment received
    </ResponseField>

    <ResponseField name="metadata" type="object">
      Custom metadata
    </ResponseField>

    <ResponseField name="created_at" type="string">
      Creation timestamp
    </ResponseField>
  </Expandable>
</ResponseField>

## Error Codes

| Code | Description |
|------|-------------|
| `ACCOUNT_NOT_FOUND` | Virtual account with this ID does not exist |
| `UNAUTHORIZED` | Invalid API key or insufficient permissions |
