---
title: 'Get Wallet'
api: 'GET /api/v1/wallets/fiat/{walletId}'
description: 'Retrieve details of a specific fiat wallet'
---

## Path Parameters

<ParamField path="walletId" type="string" required>
  The unique identifier of the wallet
</ParamField>

## Request

<ParamField header="x-api-key" type="string" required>
  API Key for authentication
</ParamField>

## Request Example

```bash
curl https://api-staging.fossapay.com/api/v1/wallets/fiat/wal_abc123xyz \
  -H "x-api-key: fp_live_sk_xxxxxxxx"
```

```javascript
const response = await fetch('https://api-staging.fossapay.com/api/v1/wallets/fiat/wal_abc123xyz', {
  headers: {
    'x-api-key': 'fp_live_sk_xxxxxxxx'
  }
});

const data = await response.json();
const wallet = data.data;
```

```python
import requests

url = "https://api-staging.fossapay.com/api/v1/wallets/fiat/wal_abc123xyz"
headers = {"x-api-key": "fp_live_sk_xxxxxxxx"}

response = requests.get(url, headers=headers)
data = response.json()
wallet = data["data"]
```

## Error Codes

| Code | Description |
|------|-------------|
| `WALLET_NOT_FOUND` | Wallet with this ID does not exist |
| `UNAUTHORIZED` | Invalid API key or insufficient permissions |
