---
title: 'Get Wallet by ID'
api: 'GET https://api-staging.fossapay.com/v1/wallets/crypto/{walletId}'
description: 'Retrieve crypto wallet details by wallet ID'
---

## Path Parameters

<ParamField path="walletId" type="string" required>
  Crypto wallet ID (crypto address entity ID)
</ParamField>

## Headers

<ParamField header="x-api-key" type="string" required>
  API Key for authentication
</ParamField>

## Request Example

```bash
curl -X GET https://api-staging.fossapay.com/v1/wallets/crypto/user:u3d \
  -H "x-api-key: your_api_key_here"
```

```javascript
const wallet = await client.cryptoWallets.getById('user:u3d');
```
