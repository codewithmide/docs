---
title: 'Get All Wallet Transactions'
api: 'GET https://api.fossapay.com/v1/wallets/crypto/{walletId}/transactions'
description: 'Retrieve all transactions for a crypto wallet'
---

## Path Parameters

<ParamField path="walletId" type="string" required>
  Crypto wallet ID (crypto address entity ID)
</ParamField>

## Headers

<ParamField header="x-api-key" type="string" required>
  API Key for authentication
</ParamField>

## Query Parameters

<ParamField query="page" type="number" default="1">
  Page number for pagination
</ParamField>

<ParamField query="limit" type="number" default="10">
  Number of items per page
</ParamField>

<ParamField query="transactionType" type="string">
  Filter by transaction type: `DEPOSIT`, `WITHDRAWAL`, `TRANSFER`
</ParamField>

<ParamField query="startDate" type="string">
  Start date filter (ISO string)
</ParamField>

<ParamField query="endDate" type="string">
  End date filter (ISO string)
</ParamField>

## Request Example

```bash
curl -X GET "https://api.fossapay.com/v1/wallets/crypto/user:u3d/transactions?page=1&limit=10&transactionType=DEPOSIT" \
  -H "x-api-key: your_api_key_here"
```

```javascript
const transactions = await client.cryptoWallets.getTransactions('user:u3d', {
  page: 1,
  limit: 10,
  transactionType: 'DEPOSIT',
  startDate: '2024-01-01T00:00:00Z',
  endDate: '2024-12-31T23:59:59Z'
});
```
