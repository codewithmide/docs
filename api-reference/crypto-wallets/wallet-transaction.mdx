---
title: 'Get Wallet Single Transaction'
api: 'GET https://api.fossapay.com/v1/wallets/crypto/transactions/{transactionId}'
description: 'Retrieve a single crypto wallet transaction by transaction ID'
---

## Path Parameters

<ParamField path="transactionId" type="string" required>
  Transaction ID
</ParamField>

## Headers

<ParamField header="x-api-key" type="string" required>
  API Key for authentication
</ParamField>

## Request Example

```bash
curl -X GET https://api.fossapay.com/v1/wallets/crypto/transactions/85503565-cb04-4ba7-abfb-b75193553086 \
  -H "x-api-key: your_api_key_here"
```

```javascript
const transaction = await client.cryptoWallets.getTransactionById('85503565-cb04-4ba7-abfb-b75193553086');
```
