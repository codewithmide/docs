Welcome to the FossaPay API documentation.

FossaPay is a payment infrastructure built for neobanks, fintechs, and businesses that need to move money quickly and safely. Through a single API, you can issue virtual accounts, manage wallets, automate payouts, and settle funds in fiat or stablecoins.

This documentation explains how FossaPay works, what you can build with it, and how to get started from your first API call to live transactions.

### What is FossaPay?

FossaPay is a unified payment and wallet infrastructure that helps products launch and scale faster.

With FossaPay, you can:
- Create and assign fiat virtual accounts on demand
- Accept bank transfers and wallet payments
- Hold and manage balances in NGN and stablecoins
- Automate payouts and settlements
- Track transactions in real time
- Receive instant webhook events
- Run KYC and compliance flows

## Core Features

### Collections API
Accept payments through fiat wallets with instant webhook notifications on every transaction.

### Payouts API
Send funds to any Nigerian bank account in seconds. Support for single and bulk disbursements with automated processing.

### Wallet Infrastructure
Create customer wallets, hold balances in NGN and stablecoins, and manage ledgers with ease.

### Compliance & Security
KYC-ready, PCI-DSS compliant, and built for high availability with 99.9% uptime guarantee.
