## Branding Customization

### Brand Identity

Configure your brand appearance across all customer touchpoints:

```javascript
await fossapay.whitelabel.setBranding({
  // Brand Name
  brand_name: 'YourBank',
  display_name: 'YourBank - Digital Banking Made Simple',

  // Colors
  colors: {
    primary: '#0080FF',
    secondary: '#00C9A7',
    accent: '#FF6B6B',
    success: '#51CF66',
    warning: '#FFC107',
    error: '#FF5252'
  },

  // Logos
  logo: {
    light: 'https://cdn.yourneobank.com/logo-light.png',
    dark: 'https://cdn.yourneobank.com/logo-dark.png',
    icon: 'https://cdn.yourneobank.com/icon.png',
    wordmark: 'https://cdn.yourneobank.com/wordmark.png'
  },

  // Fonts
  typography: {
    primary_font: 'Inter',
    secondary_font: 'Poppins',
    font_url: 'https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700'
  }
});
```

### Virtual Account Branding

Customize how virtual account names appear on bank statements:

```javascript
await fossapay.whitelabel.setAccountNaming({
  format: 'YOURBANK - {customer_name}',
  max_length: 50,
  uppercase: true,

  // Examples:
  // YOURBANK - JOHN DOE
  // YOURBANK - JANE SMITH
});
```

### Custom Domain

Use your own domain for dashboard and APIs:

```javascript
await fossapay.whitelabel.setDomain({
  dashboard: 'dashboard.yourneobank.com',
  api: 'api.yourneobank.com'
});
```

## Communication Customization

### Email Templates

Customize all customer emails:

```javascript
await fossapay.whitelabel.setEmailTemplates({
  // Welcome Email
  welcome: {
    subject: 'Welcome to {{brand_name}}!',
    from_name: '{{brand_name}} Team',
    from_email: 'hello@yourneobank.com',
    reply_to: 'support@yourneobank.com',
    template: `
      <h1>Welcome to {{brand_name}}!</h1>
      <p>Hi {{customer_name}},</p>
      <p>Your account has been created successfully.</p>
      <p>Your account number: <strong>{{account_number}}</strong></p>
    `
  },

  // Payment Received
  payment_received: {
    subject: 'Payment Received - ₦{{amount}}',
    template: `
      <h2>Payment Received</h2>
      <p>Your account has been credited with ₦{{amount}}</p>
      <p>New balance: ₦{{new_balance}}</p>
    `
  },

  // Transfer Completed
  transfer_sent: {
    subject: 'Transfer Completed',
    template: `
      <h2>Transfer Successful</h2>
      <p>₦{{amount}} has been sent to {{recipient_name}}</p>
      <p>Reference: {{reference}}</p>
    `
  },

  // KYC Approved
  kyc_approved: {
    subject: 'Identity Verified ✓',
    template: `
      <h2>Verification Complete!</h2>
      <p>Your identity has been verified.</p>
      <p>You can now access all features.</p>
    `
  }
});
```

## Feature Customization

### Enable/Disable Features

```javascript
await fossapay.whitelabel.configureFeatures({
  // Core Features
  virtual_accounts: true,
  wallets: true,
  p2p_transfers: true,
  bank_transfers: true,

  // Additional Features
  bill_payments: true,
  airtime_purchase: true,
  savings: true,
  investments: false,

  // Upcoming Features
  virtual_cards: false, // Coming Q2 2024
  physical_cards: false,
  loans: false,

  // Advanced Features
  multi_currency: true,
  stablecoins: true,
  crypto_integration: false
});
```

### Customize Transaction Limits

```javascript
await fossapay.whitelabel.setLimits({
  // Per KYC Tier
  tier1: {
    daily_transfer_limit: 50000,
    single_transaction_limit: 20000,
    monthly_volume: 300000,
    max_balance: 300000
  },
  tier2: {
    daily_transfer_limit: 200000,
    single_transaction_limit: 100000,
    monthly_volume: 1000000,
    max_balance: 1000000
  },
  tier3: {
    daily_transfer_limit: 5000000,
    single_transaction_limit: 1000000,
    monthly_volume: null, // unlimited
    max_balance: null
  },

  // Special limits
  p2p_transfer_limit: 100000,
  bill_payment_limit: 50000,
  withdrawal_daily_limit: 500000
});
```

### Fee Configuration

```javascript
await fossapay.whitelabel.setFees({
  // Transfer fees
  internal_transfer: {
    fee: 0, // Free for P2P within your platform
    fee_bearer: 'customer'
  },

  bank_transfer: {
    fee: 50,
    fee_cap: 100,
    fee_bearer: 'customer', // or 'platform'
    free_transfers_per_month: 5 // First 5 free
  },

  // Withdrawal fees
  withdrawal: {
    percentage: 0.5,
    minimum_fee: 25,
    maximum_fee: 100
  },

  // Bill payment fees
  bill_payment: {
    percentage: 1,
    cap: 100
  },

  // Account maintenance
  monthly_maintenance: 0, // Free
  dormancy_fee: 50 // After 6 months of inactivity
});
```

## User Interface Customization

### Dashboard Widgets

Customize what users see on their dashboard:

```javascript
await fossapay.whitelabel.setDashboardLayout({
  widgets: [
    {
      type: 'balance_card',
      position: 1,
      config: {
        show_currency_breakdown: true,
        show_pending_balance: true
      }
    },
    {
      type: 'quick_actions',
      position: 2,
      actions: ['transfer', 'pay_bill', 'add_money']
    },
    {
      type: 'recent_transactions',
      position: 3,
      config: {
        limit: 10,
        show_filters: true
      }
    },
    {
      type: 'savings_overview',
      position: 4,
      visible: true
    }
  ]
});
```

### Custom Fields

Add custom fields to customer profiles:

```javascript
await fossapay.whitelabel.setCustomFields({
  customer: [
    {
      field_name: 'referral_code',
      field_type: 'string',
      required: false,
      visible_to_customer: true
    },
    {
      field_name: 'preferred_language',
      field_type: 'select',
      options: ['en', 'yo', 'ig', 'ha'],
      default: 'en'
    },
    {
      field_name: 'marketing_consent',
      field_type: 'boolean',
      default: false
    }
  ]
});
```

## Compliance Customization

### KYC Requirements

Customize KYC requirements per tier:

```javascript
await fossapay.whitelabel.setKYCRequirements({
  tier1: {
    required_fields: ['email', 'phone'],
    verification_method: 'otp',
    auto_approve: true
  },
  tier2: {
    required_fields: ['bvn', 'date_of_birth', 'address'],
    verification_method: 'bvn_lookup',
    auto_approve: true,
    manual_review: false
  },
  tier3: {
    required_fields: ['government_id', 'selfie', 'proof_of_address'],
    verification_method: 'document_verification',
    auto_approve: false,
    manual_review: true,
    required_documents: ['nin', 'passport', 'drivers_license'] // any one
  }
});
```

### Transaction Monitoring Rules

```javascript
await fossapay.whitelabel.setMonitoringRules({
  // Auto-flag rules
  auto_flag: [
    {
      rule: 'large_transaction',
      threshold: 500000,
      action: 'flag_for_review'
    },
    {
      rule: 'rapid_transactions',
      count: 10,
      timeframe: '10m',
      action: 'temporary_freeze'
    },
    {
      rule: 'unusual_hour',
      hours: [0, 1, 2, 3, 4],
      threshold: 100000,
      action: 'flag_for_review'
    }
  ],

  // Auto-block rules
  auto_block: [
    {
      rule: 'sanctioned_account',
      action: 'block_and_report'
    },
    {
      rule: 'duplicate_bvn',
      action: 'block_account'
    }
  ]
});
```

## Reporting Customization

### Custom Reports

```javascript
await fossapay.whitelabel.setReportingConfig({
  daily_summary: {
    enabled: true,
    recipients: ['finance@yourneobank.com'],
    time: '06:00',
    timezone: 'Africa/Lagos',
    include: [
      'total_deposits',
      'total_withdrawals',
      'active_users',
      'transaction_volume',
      'revenue'
    ]
  },

  weekly_report: {
    enabled: true,
    day: 'monday',
    include: [
      'user_growth',
      'transaction_trends',
      'top_users',
      'failed_transactions'
    ]
  },

  custom_metrics: [
    {
      name: 'revenue',
      formula: 'SUM(transaction_fees) - SUM(wholesale_costs)'
    },
    {
      name: 'active_users',
      formula: 'COUNT(DISTINCT users WITH transactions IN LAST 30 DAYS)'
    }
  ]
});
```
